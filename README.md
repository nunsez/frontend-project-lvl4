## Simple web application - analogue of Slack chat

### [Chat link](https://serene-refuge-53938.herokuapp.com/)

#### Technologies used:
- Javascript (ES6+)
- React
- React Bootstrap
- Redux Toolkit
- REST API
- WebSocket
- Webpack
- Heroku deploy
- Rollbar
- i18next

### Install
```sh
git clone https://github.com/nunsez/frontend-project-lvl4.git
cd frontend-project-lvl4/
make install
```

#### Run locally
```sh
make start
# open in browser: http://localhost:5000/
```
